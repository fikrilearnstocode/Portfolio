# New York Housing Market Analysis

Regression analysis on New York housing market data.

## Overview

Predictive modeling and exploratory analysis on New York housing market data. Covers data cleaning, feature engineering, and regression modeling to predict house prices.

## Tech Stack

- Python, Pandas, NumPy
- Scikit-learn / XGBoost
- Matplotlib, Seaborn
- kagglehub

## How to Run

```bash
pip install kagglehub pandas scikit-learn xgboost matplotlib seaborn
jupyter notebook "New-York-Housing-Market.ipynb"
```

> Requires a Kaggle API key at `~/.kaggle/kaggle.json`
