# COVID-19 Indonesia Analysis

Time-series and provincial analysis of COVID-19 in Indonesia.

## Overview

Exploratory data analysis of COVID-19 spread across Indonesian provinces. Includes national trend visualization, provincial comparison, and 7-day rolling average smoothing to reveal wave patterns and regional disparities.

## Dataset

[COVID-19 Indonesia](https://www.kaggle.com/datasets/hendratno/covid19-indonesia) via Kaggle

## Tech Stack

- Python, Pandas, NumPy
- Matplotlib, Seaborn
- kagglehub

## How to Run

```bash
pip install kagglehub pandas matplotlib seaborn
jupyter notebook covid19-indonesia-analysis.ipynb
```

> Requires a Kaggle API key at `~/.kaggle/kaggle.json`

## Key Findings

- Java provinces (DKI Jakarta, East Java, West Java) consistently lead in case counts
- Multiple distinct waves visible in national time-series data
- 7-day rolling average effectively smooths daily reporting noise
