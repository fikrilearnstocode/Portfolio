# E-Commerce Sales Analytics

Exploratory data analysis of Brazilian e-commerce orders.

## Overview

Comprehensive EDA on 100K+ orders from the Olist Brazilian E-Commerce dataset. Identifies top revenue categories, regional sales patterns, and delivery performance trends using Python and SQL-style queries.

## Dataset

[Brazilian E-Commerce (Olist)](https://www.kaggle.com/datasets/olistbr/brazilian-ecommerce) via Kaggle

## Tech Stack

- Python, Pandas, NumPy
- Matplotlib, Seaborn
- kagglehub

## How to Run

```bash
pip install kagglehub pandas matplotlib seaborn
jupyter notebook ecommerce-sales-analytics.ipynb
```

> Requires a Kaggle API key at `~/.kaggle/kaggle.json`

## Key Findings

- Health & beauty, watches & gifts are top revenue categories
- Average delivery time: ~12 days
- Late delivery rate: ~7.5% of orders
